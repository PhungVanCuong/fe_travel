## 🎯 Data Flow

```
┌────────────────────┐
│  User Input Date   │
└─────────┬──────────┘
          │
          ↓
┌────────────────────────────────┐
│  Click "Cập Nhật" Button       │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  Show Loading Overlay          │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  POST Request to API           │
│  + Bearer Token                │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  API Process & Return Data     │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  Calculate Summary Stats       │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  Update Summary Cards          │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  Render Chart with Chart.js    │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  Update Data Table             │
└─────────┬──────────────────────┘
          │
          ↓
┌────────────────────────────────┐
│  Hide Loading Overlay          │
│  Show Toast Success            │
└────────────────────────────────┘
```
